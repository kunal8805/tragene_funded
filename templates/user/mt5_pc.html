MT5 Windows Complete Setup Guide

Follow these steps to download and login to MT5 on Windows.

Download → Install → Credentials → Login → Trade
01

Download MT5 for Windows

Download the latest MetaTrader 5 installer from the official website.

⬇ Download MT5
⚠️ Always download from the official MetaTrader website for security.
02

Install MT5

Run the installer and follow the setup wizard:

  1. Open the installer file
  2. Click Next
  3. Wait for installation to complete
  4. Open MT5
03

Get Your Trading Credentials

After purchasing a challenge, your MT5 credentials will automatically be sent to your verified email.

You will receive:

Login ID 12345678
Password ********
Server Funded-Live01
📧
Please check:
  • • Inbox
  • • Spam folder
  • • Promotions folder

If you do not receive your credentials:

📧 Contact Support

Support email: tragene.co@gmail.com

04

Login To MT5

Open MT5 and login with your credentials:

  1. Open MT5
  2. Click File in the top menu
  3. Click Login to Trade Account
  4. Enter your Login ID
  5. Enter your Password
  6. Select your Server
  7. Press Login
05

Verify Connection

Check the connection status in the bottom-right corner of MT5:

🟢 45 kb 🟢 120 kb 🔴 No Connection
⚠️
Troubleshooting:
  • • Verify your internet connection
  • • Verify your password is correct
  • • Verify you selected the correct server
  • • Restart MT5 and try again

❓ Frequently Asked Questions

Q I cannot login

Verify your login ID, password and exact server name. Ensure caps lock is off and no extra spaces are entered.

Q MT5 says no connection

Check your internet connection and restart MT5. If using a firewall, ensure MT5 is allowed through.

Q Server not found

Enter the exact server name received in your email. Type it slowly and select from the dropdown list.

Q Charts not loading

Open Market Watch (Ctrl+M), right-click, and choose Show All to display all available symbols.

Q Where are my credentials?

Credentials are automatically sent to your verified email after challenge purchase. Check inbox, spam, and promotions folders.
